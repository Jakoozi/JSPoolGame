1. The Assets Folder

The assets folder contain all the sounds and images(sprites) used in the game. It is split up into the SOUNDS and the SPRITES folder.
The sounds folder contains all the sounds used in the game. This includes the collision sounds, intro soound, whistle sounds e.t.c.
The sprites folder contains all the images used in the game. This images include the sticks, balls, pool tables and menu buttons.

2. The CSS Folder

The css folder contains the body defintion styling code for the game.

3. The Documentaion Folder

The documentation folder contains a comprehensive documentation of all the functionalities in the game. The files contained in this folder are created as html files. This is so that this files can be viewed as web pages.
